1. MVC
	Model - 순수 자바
		DTO, DAO, Service 로직 등 ...
	Controller - Servlet
		*.java
		http 통신 규약에 맞게 개발
		
	View - html/jsp
		*.html / *.jsp
		html - java 코드 사용 불가
		jsp - java 코드 사용 가능
		
		ArrayList에 저장된 데이터를 브라우저 화면에 출력시 jsp로만 가능

2. http란?
	wep 상에서 server와 client의 통신 규약
	w3c 에서 제시한 스펙
	web application들은 end user들과 소통시 사용하는 규약
	모든 프로그램 언어들은 http 프로토콜 기반의 개발을 쉽게 할 수 있게 스펙을 제시
	- 개발자들은 스펙에 맞게 개발(http 지원하는 API 제공)
