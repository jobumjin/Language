* JRE(Java Runtime Environment)
    * JVM + API
    * 실행 환경
    
* JDK(Java Development Toolkit)
    * Compiler 를 비롯한 개발에 필요한 여러도구 + JRE(실행 도구)
    * Jdk/bin

자바 소프트웨어 개발 process
개발 -> 컴파일 -> 실행

Java Platform 두가지 구성요소
* Java Virtual Machine(JVM,자바가상머신, os에 종속적)
	* os에 적합하게 번역(컴파일)된 것을 실행 하는 실 주체의 기능
	* os에 종속적인 JDK를 설치했어야 함
* Java Application Programming Interface (API)

1. 프로개름 개발 사유
	- 사용자(end user, client)유입
	- 사용자들에게 최적화된 서비스제공
	- 개발 및 서비스를 위한 고려 사항
		1. 요구사항 파악
		2. 서비스 구체화, 시스템 구축 및 설계
			- 장비 & 개발 일력 등...
		3. 개발 완료 후 서비스
		...	
	- 프로그램에 있어서 가장 중요한 것?
		1. 데이터
			- 변수
				- 타입이 다양
					문자, 문자열, 실수, 정수, 논리 ...
					
		2. 데이터를 어떻게 처리 할 지에 대한 기능(서비스)
			- 메소드(함수)
				- 예시
					- 계산기 기능(사칙연산만 가능한 단순 계산기)
						- 숫자값 저장 가능한 변수(메모리)
						- 사칙 연산용 기능의 메소드(4개..)
						사칙연산,그 외 다양한 기능
						수행 : 연산을 위한 값들(실수? 정수?)

2. 필수 암기 사항
	1. 소스 구조
		1. 확장자 - *.java
		2. java 소스의 구성
			package 선언구
				- 현 소스의 저장 경로
				- package명 디렉토리 구조 및 이름
			
			import 선언구
				- 다른 package의 프로그램을 가져와서 사용하겠다는 의미
			
			class 이름 {
				변수(데이터)
				생성자(객체생성시에 호출되는 필수 구성 요소)
				메소드(기능)
			}
			
			
	2. main(메소드)
